Band

The band element is full-width, stackable, has no margin-bottom and can have a background colour applied (for example see, coloured bands).

The band is used to stack together layers of page content.

The default band has a white background.

The mark up consists of a parent element with a class of "band", containing a child element with a class of "band--inner-container".

The inner container is used to set the vertical padding and constrain the content to a width.
