Tabs

On redgate.com tabbed sections of content have a "lozenge" style navigation.

Tabbed content is normally marked up within a band and, other than the default white, you can use these colour schemes: lightest-grey, lightest-blue, darkest-grey and darkest-blue.

Use the Tabs JS component to add the functionality to the tabs.
